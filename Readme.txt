This unofficial tool will automatically upload the posts on Instagram as per scheduled time.

Features :
	1. Uses selenium web browser
	2. Uses scheduler pkg to schedule the job
	3. Videos can be posted


Limits :
	1. Insta changes the xpath, the same changes need to be updated in the script.
	2. Username and Passwords are saved plain text.
	3. You will have to keep updating selenium webdriver with new chrome release version.
